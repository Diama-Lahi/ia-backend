# IA Backend (Flask + OpenAI)

## Endpoints
- POST /chat
- POST /image-edit
- POST /image-variation

## Déploiement Render
1. Créer un repo GitHub vide
2. Ajouter ces fichiers
3. Sur Render → New Web Service
4. Connecter le repo
5. Ajouter la variable d'environnement :
   - OPENAI_API_KEY = ta clé OpenAI
6. Deploy
